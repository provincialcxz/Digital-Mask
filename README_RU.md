# [FSOCIETY] Digital-Mask: User-Agent Switcher

> *"Hello, friend."*  

> [Read in English](README.md)

Расширение для Google Chrome для полной подмены цифрового отпечатка браузера — User-Agent, Client Hints, JS-свойств, Canvas, WebGL и защиты от утечки IP через WebRTC.



## Возможности

### User-Agent подмена
- Подмена `User-Agent` заголовка на сетевом уровне.
- Одновременная подмена всех **Client Hints** заголовков.
- Подмена `userAgentData` в JS-контексте страницы.
- Поддержка Chrome, Edge, Opera, Firefox, Safari, Googlebot — каждый браузер имитируется корректно.

### База профилей
- Встроенная база ЮА-строк.
- Фильтрация по категориям: **Desktop / Mobile / Bots / Saved**
- Кнопка случайного ЮА из активной категории

### Свои профили
- Ввод произвольного ЮА вручную
- Автосохранение
- Список сохранённых профилей с быстрым выбором

### Правила для сайтов
- Разные ЮА для разных сайтов: `google.com`, `*.youtube.com`, `*tracker*`
- Правила работают с приоритетом выше глобального.
- Включение/отключение каждого правила по отдельности.
- Паттерны автоматически конвертируются в валидный DNR.

### Защита приватности (вкладка PRIVACY)

| Функция | Описание |
|---|---|
| **WebRTC Leak Protection** | Блокирует утечку реального IP через WebRTC |
| **JS Fingerprint** | Подменяет стандартные параметры|
| **Canvas Noise** | Добавляет субпиксельный шум — ломает canvas fingerprint |
| **WebGL Spoof** | Скрывает GPU vendor/renderer, заменяет на типичные значения Intel |
| **Timezone Sync** | Синхронизирует TimeZone с реальным IP |

### Импорт / Экспорт
- Экспорт всей базы в `.txt` (один ЮА на строку) или `.json` (с правилами сайтов).
- Импорт из `.txt` и `.json` с дедупликацией.
- JSON-экспорт включая правила для сайтов.

### Интерфейс
- Переключение языка **EN / RU**
- Индикатор активности спуфинга
- Отображение реального ЮА браузера



## Установка

**Предпочтительный вариант:**

Перейти на страницу расширения в [Google Chrome]().  


**Локальная установка:**  
1. Скачать или клонировать репозиторий
2. Открыть `chrome://extensions`
3. Включить **Режим разработчика** 
4. Нажать **Загрузить распакованное расширение**
5. Выбрать папку с расширением



## Разрешения

| Разрешение | Зачем |
|---|---|
| `declarativeNetRequest` | Модификация HTTP-заголовков |
| `declarativeNetRequestWithHostAccess` | Доступ ко всем хостам |
| `storage` | Сохранение настроек и профилей |
| `privacy` | Управление WebRTC политикой |



## Ограничения

- **IP-репутация**: Если ваш IP находится в базах VPN/datacenter провайдеров (MaxMind и др.) — расширение не поможет. Это определяется на стороне сервера.
- **Timezone для Date**: `Date.prototype` методы зависят от системного timezone ОС — расширение патчит только `Intl.DateTimeFormat`, что покрывает большинство детекторов.
- **После применения настроек PRIVACY**: необходимо перезагрузить активные вкладки.
